# Raw Strings

To preserve all characters including special characters, add # to the beginning and end of a string literal. Must be placed outside of the string's double quotes.

let myString = #"The quick brown fox jumped \n\n\nover the lazy dog"#

# String Interpolation

String interpolation is a way to assemble a string using variables, constants, expressions and literals. To use interpolation, items are wrapped in \() inside a string literal or multiline string literal as a placeholder.

let myName = "Jane Smith"
print("Hello, my name is \(myName). Nice to meet you!")

// outputs Hello, my name is Jane Smith. Nice to meet you!

# Comparing Strings

String and character comparisons are checked using the the 'equal to'== and 'not equal to' != comparison operators

let string1 = "The quick brown fox..."
let string2 = "The quick brown fox..."

print(string1 == string2)   // true
print(string1 != string2)   // false
